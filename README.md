# Snake Game

一个经典贪食蛇（Snake）小游戏的需求与原型说明仓库。

本仓库当前包含：
- `snake/Snake-Game-PRD.md`：贪食蛇小游戏需求文档（用于指导后续开发实现）

---

## 1. 游戏简介

玩家控制一条不断移动的蛇在网格地图中前进，通过吃食物获得分数并变长。蛇越长越难操控，最终目标是获得更高分数。

---

## 2. 基本规则

### 2.1 地图
- 游戏在一个二维网格地图中进行（默认 20×20，可配置）。
- 地图边界为墙，默认不可穿越。

### 2.2 移动
- 蛇会按固定节奏（tick）自动向当前方向前进一格。
- 玩家只能改变蛇的移动方向，不能让蛇停止（除非暂停）。
- 不允许 180° 直接反向（例如向右时不能立刻改为向左）。

### 2.3 吃食物
- 蛇吃到食物后：
  - 蛇长度 +1
  - 分数增加（默认 +10）
  - 食物会在地图上随机位置重新生成（不会生成在蛇身上）

### 2.4 失败条件（Game Over）
满足任一条件，游戏结束：
- 蛇头撞墙
- 蛇头撞到自己的身体
- （若启用）撞到障碍物

### 2.5 速度变化
- 随着分数提升，蛇移动速度会逐步加快（动态加速）。

---

## 3. 道具（可选能力）

在需求文档的 v1 版本中，建议至少实现 1 种道具（也可实现 2 种）：

- **双倍得分（2x Score）**：短时间内吃食物得分翻倍
- **减速（Slow）**：短时间内降低蛇移动速度

道具通常随机生成，且不会与蛇身或食物重叠。

---

## 4. 操作说明（建议默认键位）

### PC
- 方向键：↑ ↓ ← →
- 或 WASD：W A S D
- 空格：暂停/继续
- Enter：开始/重开

> 移动端（如需支持）可使用滑动手势控制方向。
